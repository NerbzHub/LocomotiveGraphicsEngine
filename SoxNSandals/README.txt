Sox N Sandals is an OpenGL application that has the capacity to
	import OBJ models and light them with phong lighting.

The name is inspired by the AIE Bootstrap.

The source files are found in Source/SoxNSandals.
The release build is in Build/Release.

Controls:

Movement:
	Forward:	W
	Left:		A
	Back:		S
	Right:		D
	Sprint/Run:	Left Shift

Look around:
	Mouse Movement

Reference Sources:
	Crytek Sponza Model sourced from: http://www.crytek.com/cryengine/cryengine3/downloads
	Soul Spear sourced from AIE's Canvas Student Portal.
	The classes Texture, OBJMesh and, Shader were given to me from AIE to assist my work.
	Files that aren't mine nor AIE: gl_core_4_5.c, gl_core_4_5.h, stb_image.h, tiny_obj_loader.h
	Tutorials, Presentations and Teachers from AIE also assisted me.
